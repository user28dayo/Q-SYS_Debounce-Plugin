# Q-SYS Debounce Filter Plugin

Q-SYS上で動作する、信号のチャタリング（バタつき）を防止・吸収するためのユーティリティプラグインです。

ネットワーク機器のステータス監視などで、瞬間的に `true` と `false` が切り替わってしまうような不安定な入力信号を、指定した時間（Delay）だけ待つことで安定した出力に変換します。

## 🌟 Features (特徴)

* **チャタリングの確実な吸収**: 入力信号が連続してバタついた場合、変化が収束するまで出力を保留します。
* **柔軟な時間設定**: 0秒から60秒まで、任意の待機時間（DelayTime）を自由に設定可能です。
* **軽量でシンプルな設計**: 不要な処理を省き、システムに負荷をかけずに動作します。
* **使いやすいフォルダ階層**: プラグインリスト内で `user28dayo` フォルダにスッキリと格納されます。

## 📦 Installation (インストール方法)

1. このリポジトリから `Debounce Filter.qplug` ファイルをダウンロードします。
2. ダウンロードしたファイルを、以下のQ-SYS Designerのプラグインフォルダに配置してください。
   * **Windows:** `Documents\QSC\Q-SYS Designer\Plugins`
3. Q-SYS Designerを再起動するか、F5キーを押してプラグインを再読み込みします。
4. 右側の `Schematic Elements` ペインの `Plugins` > `user28dayo` フォルダから、本プラグインをワークスペースにドラッグ＆ドロップして使用します。

## 🛠 Usage (使い方)

プラグインには以下の3つのコントロール/ピンがあります。

* **In (Input)**
  * 監視したい信号（ネットワーク機器のステータスや物理ボタンの入力など）を接続します。
* **Out (Output)**
  * チャタリングが吸収され、安定化された信号が出力されます。後段のロジックや制御に接続してください。
* **Delay(s)**
  * 信号が変化してから「安定した」と判定するまでの待機時間（秒）を設定します。
  * デフォルトは `0.5秒` です。`0` に設定した場合は遅延なし（即時反映）として動作します。

## 📝 Example Scenario (活用例)

* **プロジェクターの電源ステータス監視:**
  起動時やステータス遷移時にステータスが不安定に切り替わる機器に対し、このフィルターを挟むことで、UI上のインジケーターが点滅するのを防ぎます。
* **物理接点のノイズ除去:**
  GPIなどで受け取ったリレーの物理的なチャタリングノイズをソフトウェア的に除去します。
